# Giới thiệu

Chào mừng bạn đến với hệ thống tài liệu chính thức của **TriAnh Solutions** — giải pháp liên lạc số, tổng đài và chăm sóc khách hàng đa kênh dành cho doanh nghiệp.

**TriAnh cung cấp:** tổng đài ảo (CCMS), Voice Brandname, IVR, SMS Brandname, Zalo OA/ZNS, TCRM (hệ thống CRM tích hợp kênh thoại), API & SDK cho tích hợp nhanh.
