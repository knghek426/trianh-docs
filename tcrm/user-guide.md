# Hướng dẫn sử dụng nhanh (TCRM)

## Đăng nhập
- URL: `https://tcrm.trianh.vn`
- Quyền: Admin / Agent / Viewer

## Quy trình nhận cuộc gọi (Agent)
1. Agent đăng nhập -> vào tab `Calls`.
2. Khi có cuộc gọi, cửa sổ contact sẽ bật ra (nếu tìm thấy contact).
3. Ghi chú, chuyển ticket, hoặc tạo task follow-up.

## Xuất báo cáo
- Vào Dashboard -> Chọn khoảng thời gian -> Export CSV/PDF
